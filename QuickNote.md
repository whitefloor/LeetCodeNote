# QuickNote
整理 Grid169 內出現的經典題目出現過的算法筆記  
目的是幫助在面試之前可以快速複習必刷算法

# Hash Map
## 核心概念
1. Key-Value 資料結構
## 場景
1. Union Find
> 找子集的共同父集時壓縮路徑時會使算法時間複雜度大幅減少
## 註記
通常在刷提過程中不太會需要用到，通常大部分情況下可用 Array 替代  
僅在不確定資料範圍、無法計算 Array Index 的時候才會使用  
不然通過時間和 Memory 都輸別人一大截

# Stack
## 核心概念
1. Last In, First Out  
2. 由 Array 實現
## 場景
1. Polish notation（波蘭表示法、前綴表示法）
> 計算加減乘除及括號會使用到，十分重要
